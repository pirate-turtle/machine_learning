# 프로그래머스 과제
[머신러닝] 채용 공고 추천  
<https://programmers.co.kr/skill_check_assignments/1>

## 문제 개요
개발자(userID로 표현됨)와 채용공고(jobID로 표현됨)를 보고 개발자가 해당 채용공고에 지원할지 안 할지를 예측하는 Binary Classifier를 만들 것.
이 모델을 만드는데 필요한 feature들은 부가적으로 제공되는 데이터 파일들을 바탕으로 만들어내면 된다.
